Conéctese directamente a su Tesla Wall Connector para supervisar el consumo de energía y el estado del cargador. Esta aplicación funciona con el Tesla Wall Connector Gen 3 y el Universal Wall Connector.

El TWC no se puede controlar. El uso compartido de energía (Power Sharing) solo proporciona acceso al TWC principal; los datos de las unidades esclavas no son accesibles.

La aplicación utiliza la detección automática mDNS. Al añadir un nuevo dispositivo, Homey escanea automáticamente su red en busca de Wall Connectors compatibles. Para garantizar una conectividad permanente, se recomienda encarecidamente establecer una IP estática o una reserva DHCP para su Wall Connector en la configuración del router.
